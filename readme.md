- [Simple, fast].

- Expressive, intuitive [database ORM].
- [Ability to find all products that you need].

## Steps to setup this Project

- open cmd
- go to  htdocs directory.
- git clone https://github.com/dev-fathy/website.git ProjectName
- cd ProjectName
- composer install
- php artisan key:generate
- php artisan migrate
- php artisan db:seed
